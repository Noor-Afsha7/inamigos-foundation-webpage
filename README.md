# inamigos-foundation-webpage


A responsive website created for **InAmigos Foundation**, a non-profit organization dedicated to social welfare, education, environmental conservation, animal welfare, and community development.

## Features

* Responsive design for desktop and mobile devices
* About Us section
* Image gallery showcasing initiatives
* Projects showcase
* Impact section highlighting achievements
* Call-to-Action section
* Footer with contact information and social links
* Modern glassmorphism-inspired design

## Built With

* HTML5
* CSS3
* Flexbox
* Media Queries
